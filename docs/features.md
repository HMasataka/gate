# Gate - 認証認可サーバー機能一覧

## 認証 (Authentication)

### ユーザー登録・管理

- メールアドレス + パスワードによるユーザー登録
- メールアドレスの確認 (メール検証フロー)
- パスワードハッシュ (argon2id + crypto/rand によるソルト生成、PHC 形式で保存)
- パスワード変更
- パスワードリセット (メールによるリセットリンク発行)
- アカウントの有効化・無効化
- アカウント削除 (論理削除)

### ログイン・ログアウト

- メールアドレス + パスワードによるログイン
- セッション発行・破棄
- ログアウト (トークン失効)
- 同時セッション数の制限

### 多要素認証 (MFA)

- TOTP (Google Authenticator, Authy 等)
- WebAuthn / FIDO2 (パスキー)
- リカバリーコード

### ソーシャルログイン

- Google
- GitHub
- Apple
- 汎用 OAuth2 / OIDC プロバイダ連携

## 認可 (Authorization)

### ロールベースアクセス制御 (RBAC)

- ロールの作成・更新・削除
- ユーザーへのロール割り当て
- ロールの階層構造 (継承)

### パーミッション管理

- パーミッションの定義 (例: `users:read`, `users:write`)
- ロールへのパーミッション割り当て
- ユーザー単位のパーミッション上書き

### スコープ管理

- OAuth2 スコープの定義
- クライアントごとの許可スコープ設定
- スコープに基づくアクセス制御

## トークン管理

### アクセストークン

- JWT 形式 (RS256 / ES256)
- カスタムクレームの付与
- 有効期限の設定

### リフレッシュトークン

- Opaque トークン形式
- トークンローテーション (使用時に新しいトークンを発行)
- 有効期限の設定
- ファミリー検出 (リプレイ攻撃対策)

### トークン失効 (Revocation)

- 個別トークンの失効
- ユーザー単位の全トークン失効
- クライアント単位の全トークン失効

## OAuth 2.0

### グラントタイプ

| グラントタイプ            | ユースケース                 |
| ------------------------- | ---------------------------- |
| Authorization Code + PKCE | Web アプリ、モバイルアプリ   |
| Client Credentials        | サーバー間通信 (M2M)         |
| Device Authorization      | TV、IoT 等の入力制限デバイス |

### クライアント管理

- クライアントの登録 (client_id / client_secret 発行)
- クライアントの更新・削除
- クライアント種別 (confidential / public)
- リダイレクト URI の管理
- 許可スコープの設定

## OpenID Connect (OIDC)

### トークン・エンドポイント

- ID Token の発行 (JWT)
- UserInfo エンドポイント
- Token エンドポイント
- Authorization エンドポイント

### ディスカバリ

- `/.well-known/openid-configuration`
- `/.well-known/jwks.json`

### 標準クレーム

- `sub`, `email`, `name`, `picture` 等の標準クレームサポート
- カスタムクレームの拡張

## セキュリティ

### ブルートフォース対策

- ログイン試行のレートリミット
- 連続失敗時のアカウントロック
- CAPTCHA 連携 (閾値超過時)

### トークンセキュリティ

- JWT 署名鍵のローテーション
- トークンバインディング
- JTI (JWT ID) によるリプレイ防止

### 通信・リクエスト保護

- HTTPS 強制
- CORS 設定
- CSRF 対策 (state パラメータ検証)
- PKCE 必須化 (public クライアント)

### 入力検証

- パスワード強度ポリシー (最小文字数、複雑性要件)
- メールアドレスのフォーマット検証
- リダイレクト URI の厳密な一致検証

## 管理機能

### Admin API

- ユーザーの CRUD・検索
- ロール・パーミッションの管理
- クライアントの管理
- システム設定の変更

### 監査ログ

- ログイン・ログアウトの履歴
- 認証失敗の記録
- 権限変更の記録
- トークン発行・失効の記録
- 管理操作の記録

### モニタリング

- ヘルスチェックエンドポイント
- メトリクス公開 (Prometheus 形式)
- 認証成功率・失敗率の統計

## インフラ・運用

### データストア

- ユーザー・クライアント・ロール等の永続化 (RDB)
- セッション・トークンのキャッシュ (Redis 等)
- 鍵の安全な保管

### 設定

- 環境変数 / 設定ファイルによる構成
- トークン有効期限の設定
- パスワードポリシーの設定
- CORS 許可オリジンの設定

### デプロイ

- Docker イメージの提供
- ステートレス設計 (水平スケーリング対応)
- グレースフルシャットダウン
